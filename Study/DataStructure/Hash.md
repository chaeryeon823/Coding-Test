## Hash(해시)

파이썬에서 해시는 딕셔너리 자료형으로 사용.

### 딕셔너리와 리스트의 시간 복잡도 차이

| Operation   | Dictionary | List        |
| ----------- | ---------- | ----------- |
| Get Item    | O(1)       | O(1)        |
| Insert Item | O(1)       | O(1) ~ O(N) |
| Update Item | O(1)       | O(1)        |
| Delete Item | O(1)       | O(1) ~ O(N) |
| Search Item | O(1)       | O(N)        |
